# brikX / brix.lk

A modern Laravel-based company website for **brikX**, designed as a premium, responsive, inquiry-focused platform for construction, products, services, and built-space delivery.

The website is built to give clients one coordinated place to understand services, view products/projects, and send inquiries, while allowing all main content to be managed from the admin panel.

---

## Project Overview

**brikX** is positioned as a modern company for clients who want one coordinated partner across built-space delivery.

The website focuses on:

- Premium dark theme design
- Responsive frontend
- SEO-friendly pages
- Admin-managed website content
- Inquiry-based product flow
- Services and project showcase
- Customizable hero, sections, products, and SEO data
- Backend-powered content management

## Main Features

### Frontend Website

- Modern dark premium design
- Fully responsive layout
- Home page with hero section
- Services section
- Products section
- Projects / portfolio section
- About company section
- Contact / inquiry form
- SEO-friendly page structure
- Clean UI animations
- Three.js-ready hero section support

### Admin Panel

- Manage website pages
- Manage hero section content
- Manage services
- Manage products
- Manage projects
- Manage inquiries
- Manage SEO meta title and description
- Manage images and content
- Enable / disable website sections
- Update company details without code changes

### SEO Management

- Custom meta title
- Custom meta description
- Custom meta keywords
- Open Graph title
- Open Graph description
- Open Graph image
- Canonical URL support
- Static page SEO management
- Product and service SEO support

### Product / Inquiry Flow

- Products can be listed without public prices
- Customers can send inquiries
- Admin can review inquiries
- Secure inquiry-led sales process
- Suitable for business-to-business and project-based sales
