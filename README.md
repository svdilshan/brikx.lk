## 🏗️ Project Overview

**brikX** is a modern Laravel-based company website for **[brix.lk](https://brix.lk)** — a premium, responsive, inquiry-focused platform for construction, products, services, and built-space delivery in Sri Lanka.

> One coordinated partner across built-space delivery — construction, systems, and client needs, all in one place.

The website gives clients a single coordinated platform to understand services, view products & projects, and send inquiries — while all main content is managed through a powerful admin panel.

## 🚀 Installation Guide

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/svdilshan/brix-lk.git
cd brix-lk
```

### 2️⃣ Install PHP Dependencies

```bash
composer install
```

### 3️⃣ Install Node Dependencies

```bash
npm install
```

### 4️⃣ Environment Setup

```bash
cp .env.example .env
php artisan key:generate
```

### 5️⃣ Configure Database

Open `.env` and update your database credentials:

```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=brix_db
DB_USERNAME=your_username
DB_PASSWORD=your_password
```

### 6️⃣ Run Migrations & Seeders

```bash
php artisan migrate --seed
```

### 7️⃣ Build Frontend Assets

```bash
npm run build
```

### 8️⃣ Storage Link

```bash
php artisan storage:link
```

### 9️⃣ Start Development Server

```bash
php artisan serve
```

> 🌐 Visit `http://localhost:8000` in your browser.

## 🎛️ Admin Panel

Access the Filament admin panel at:

```
http://localhost:8000/admin
```

Create an admin user via:

```bash
php artisan make:filament-user
```

## 🌐 Deployment on cPanel (Shared Hosting)

```bash
# 1. Upload files to public_html or subdirectory
# 2. Point document root to /public
# 3. Set PHP version to 8.2+ in cPanel
# 4. Import database via phpMyAdmin
# 5. Update .env with production credentials
# 6. Run via SSH or deployment script:

php artisan migrate --force
php artisan config:cache
php artisan route:cache
php artisan view:cache
npm run build
```
